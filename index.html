Information

To use this calculator:

  1. Have a spotter determine the distance and azimuth to the target location.
  2. Without moving, have the spotter determine the distance and azimuth to the artillery location.
  3. Input the two sets of values, and the computed result is the distance and azimuth from the artillery to the target.
  4. The optional left/right and drop/add correction can be entered as viewed from the spotter in meters. Negative values indicate correction to the left/closer to the spotter.
  5. Adjust the artillery settings to match the resulting values, then fire!
  6. You can optionally insert linked Target Reference Points between the spotter and the arty, this allows for spotting beyond binocular range.

The In-game values in the result are rounded to match the in-game increments.

You can save firing plans for later use. To do this, add a new plan from the menu and enter a name. The current state will be persisted and can be reloaded by selecting it form the firing plan menu. The currently active plan is displayed and by pressing the button, you can reset the current values to the stored ones. Note that the correction portion is not restored.
